<static> createDocument(name, namespaces, prefix) → {Document}

Create a new XML DOM document.
Parameters:
Name Type Argument Description
name string The element name for the main element in the document (for example `schema`).
namespaces Object.<string, string> The namespaces to define on the main element. They key is short notation (like `xs`), the value is the namespace URI.
prefix string <optional>
 The prefix to be used before the main element's name (for example `xs` to get `xs:schema`).
